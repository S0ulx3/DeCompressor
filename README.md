# Español
# Instalación Español
---------------------

- Clonamos el repositorio
```
git clone https://www.github.com/S0ulx3/DeCompressor
```
- Otorgamos permisos de ejecución
```
chmod +x DeCompressor.sh
```
- Ejecutamos el Script
```
./DeCompressor.sh
```
